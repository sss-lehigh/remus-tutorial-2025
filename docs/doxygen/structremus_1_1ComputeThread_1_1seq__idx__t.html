Detailed Description

An object to reserve and release coroutine-local slots.

seq_idx_t provides the next available slot in the ring for a coroutine-local operation. The number of slots is determined by the CN_OPS_PER_THREAD argument. Each ComputeThread has a separate seq_idx_t for each coroutine index (coro_idx).
