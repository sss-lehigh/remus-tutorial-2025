Detailed Description

template<typename T>
class remus::rdma_ptr< T >

A "smart pointer" to memory on another machine.

Template Parameters
TThe type of the object that this pointer points to

The rdma_ptr onject encapsulates information relavant to RDMA communication in a regular pointer. In the high 16 bits, it stores the id of the remote machine, and in the low 48 bits, it stores the raw address of the object on that machine. By combining these two pieces of information, we provide a useful abstraction for subsequent RDMA operations.
