Detailed Description

A control block for managing segments in the distributed memory system.

ControlBlock is a header for each Segment managed by a Memory Node. It supports bump allocation and graceful shutdown, and offers some optional space for a barrier (for synchronizing compute threads) and root pointer (cast to rdma_ptr<> to reach the workload's root)
