Detailed Description

An object to reserve and release slots in a staging buffer for RDMA communication.

op_counter_t provides the next available slot in the staging buffer based on the ComputeThread's op_counter_end and op_counter_assignments. The number of slots is determined by the CN_OPS_PER_THREAD argument. TODO: All field names should end with an underscore
